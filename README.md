# Multimodal Machine Learning-Based Early Warning System for Student Mental Wellbeing

## Description

This project aims to develop a multimodal machine learning system for the early detection of mental health risks among students using textual, behavioral, and facial expression data.

## Tech Stack

- Frontend: React + TypeScript
- Backend: Flask (Python)
- Machine Learning: Scikit-learn
- Database: PostgreSQL
